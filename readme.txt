Written using Python 3.5.3 and Pygame 1.9.3
To launch the game, just run Main.py
The game does not launch in fullscreen by default as it does not display properly if the screen is not large enough. If you want to launch it in fullscreen, set FULLSCREEN to True (line 13 in Main). It was designed to display on a 1920x1080 (16:9 aspect ratio)
There is music. (Source: https://soundcloud.com/ovolandia/quando-a-revolucao-vier)
